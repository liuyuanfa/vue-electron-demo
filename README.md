# vue-electron-demo

## 相关工具版本
```
node: 16.15.0
yarn: 1.22.19
vue: 3.2.13
@vue/cli: 5.0.8
```


## 安装依赖
```
yarn install
```

## 安装依赖
```
yarn install
```

### 项目运行
```
yarn serve
yarn electron:serve
```

### 项目打包
```
yarn electron:build
```
